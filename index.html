    Proyectos

    Landing Page

    Esta es una landign page creada para una clinica veterinaria cuenta con 5 secciones (Galeria, Servicios, Personal, Contacto, Nosotros)

    Tecnologías utilizadas: Html | Css | Javascript

    C.R.U.D

    Este es un C.R.U.D que guarda los datos en el local Storage está enfocado en llevar el control de vacunación de una o varias mascotas, tiene opciones interesantes como la de un filtro para buscar por alguna palabra clave ya sea el nombre de la mascota, de la vacuna, o incluso de la fecha de vacunación.

    Tecnologías utilizadas: Html | Css | Javascript

    Covid-19 Dashboard

    Este es un proyecto colaborativo el cual desarrollamos con Eder Ramos en el cual consumimos dos APIs para detectar la cantidad de contagiados y de fallecidos por Covid-19 en Latinoamerica se puede hacer comparaciones entre paises y tambien se ve reflejado en graficos

    Tecnologías utilizadas: Html | Css | Javascript

    Aplicación de restaurante

    Esta aplicación esta aplicación esa enfocada en que un usuario pueda hacer una reservación a un restaurante, cuenta con una sección de login y de registro así mismo el dueño del restaurante a través de su panel de administrador pueda ver todas las reservas hechas por los diferentes usuarios ademas el administrador tambien puede cambiar el menú a través de su panel.

    Tecnologías utilizadas: Html | Css | Javascript | React | Firebase

    Api Ecommerce
